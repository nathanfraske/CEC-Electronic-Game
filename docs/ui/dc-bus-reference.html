DC BUS LIVE

Showing voltage and current on a DC belt bus

A reference for the game's visual language on a power bus. Voltage is a property of the net, so it reads as a shared level: the belt's height, its rail color, and a number, uniform along the run and sagging only under load. Current is flow along the belt, shown by direction, speed and density, and the belt's thickness. Where a load taps the bus the flow divides, which is current conservation made visible. Drag the rail loads, change the flow speed, and toggle the IR drop to see each rule respond.
