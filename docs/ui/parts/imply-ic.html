The IMPLY gate, five layers down

An IMPLY gate, written A implies B or A arrow B, gives low out in exactly one case, when A is true and B is false; every other case is high. It is the material conditional, "if A then B," which works out to NOT A, OR B, equivalently NAND(A, NOT B). This is the first gate in the set where only one input is inverted: B passes through a small inverter to make NOT B, and that feeds a NAND with A straight in. Six transistors, one hidden node Bi, and an asymmetry the others did not have, since the gate is negative-unate in A but positive-unate in B. There is no discrete IMPLY chip; in static CMOS it is just an inverter feeding a NAND, the cell shown here. Its real hardware home is memristor stateful logic, where the state is a resistance rather than a voltage and IMPLY together with FALSE forms a complete logic basis. The first tier is the logic symbol, an AND body with a bubble on the B input and a bubble on the output. The second is the gate as a flow network. The third opens it up as pressure-pilot valves. The fourth is the device as a schematic, the B inverter feeding the two-input NAND. The fifth is the silicon, the metal-oxide cross-section of all six devices on one die. Each input is a real voltage, so the logic shows at the ends and the switching event shows in the transfer curve.
