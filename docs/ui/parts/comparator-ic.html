The latched comparator, five layers down

A comparator answers one question: which input is higher. It is the 1-bit converter that turns an analog difference into a single digital bit. Here it is the ADCMP601, a rail-to-rail latched comparator in a six-pin SC70 package. The output Q goes high when the analog input VP sits above VN, and the latch control LE decides whether that result is held: drive LE low to latch the last decision, leave it high to track. The defining heart is a regenerative cross-coupled latch, a positive-feedback loop that drives a tiny input difference to a full rail in nanoseconds, the same cross-coupled cell that stores a bit in a flip-flop. The first tier is the logic symbol with the real pinout and the decision table. The second is the flow network: two cross-coupled chambers, each piloting the other to the opposite level, with the differential input tipping the balance and an LE valve that holds it. The third opens the differential pair as pressure-pilot valves, where the higher input lifts its plug more and drains its node. The fourth is the schematic, the diff pair feeding the cross-coupled latch over an LE-gated tail current source. The fifth is the silicon, the differential pair and the cross-coupled load on one die. When the two inputs are nearly equal the latch hangs near mid-rail, the metastable state, balanced and slow to decide. The metastable band shown here is widened for the eye, since the real input offset is only a few millivolts.
