The 4-input LUT, five layers down

A lookup table is a memory you address with your inputs. This is the house CEC2064, a 4-input configurable logic cell: sixteen stored configuration bits hold a truth table, the four inputs I0 to I3 form a 4-bit address, and the addressed bit drives the output Y. That is all of combinational logic in one structure, since any function of four inputs is just a different set of sixteen bits. The hardware that does the addressing is a 16:1 multiplexer tree, four stages of 2:1 selects, one per input. An optional output flip-flop registers Y on the clock edge, and a LUT followed by a flip-flop is the FPGA logic element that every programmable fabric is built from. The first tier is the symbol with the real pinout and the live truth table. The second is the flow network, the sixteen config reservoirs and the mux funnel that opens a channel from one stored bit to the output. The third opens one funnel stage as pressure-pilot valves. The fourth is the device, a bank of sixteen SRAM cells feeding a pass-transistor mux tree into the optional register. The fifth is the silicon, one SRAM config cell and one pass transistor.
