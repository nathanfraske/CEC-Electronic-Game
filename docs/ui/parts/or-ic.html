The OR gate, five layers down

An OR gate gives high out when either input is high, and low out only when both are low. Here it is one gate, the single 2-input OR chip SN74LVC1G32, in a five-pin package with inputs A and B on pins 1 and 2. There is a catch that this set makes visible: static CMOS can only build inverting gates, so a real OR is a NOR stage followed by an inverter stage, six transistors, with a hidden internal node Yi between them. The first tier is the logic symbol with the real pinout. The second is the overall analogy, the gate as a flow network: a first machine (the NOR) fills an internal chamber Yi, and that level pilots a second machine (the inverter) that fills the output chamber Y. The third opens both stages up as pressure-pilot valves, the working MOSFET analogy. The fourth is the real device, the SOT-23-5 package and the full CMOS network drawn as a schematic, the NOR four transistors driving the inverter two. The fifth is the silicon, the metal-oxide cross-section of all six devices on one die. Each input is a real voltage, so the logic shows at the ends and the two switching events show in the transfer curve.
