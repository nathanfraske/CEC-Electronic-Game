The AND gate, five layers down

An AND gate gives high out only when both inputs are high, and low out otherwise. Here it is one gate, the single 2-input AND chip SN74LVC1G08, in a five-pin package with inputs A and B on pins 1 and 2. There is a catch that this set makes visible: static CMOS can only build inverting gates, so a real AND is a NAND stage followed by an inverter stage, six transistors, with a hidden internal node Yi between them. The first tier is the logic symbol with the real pinout. The second is the overall analogy, the gate as a flow network: a first machine (the NAND) fills an internal chamber Yi, and that level pilots a second machine (the inverter) that fills the output chamber Y. The third opens both stages up as pressure-pilot valves, the working MOSFET analogy. The fourth is the real device, the SOT-23-5 package and the full CMOS network drawn as a schematic, the NAND four transistors driving the inverter two. The fifth is the silicon, the metal-oxide cross-section of all six devices on one die. Each input is a real voltage, so the logic shows at the ends and the two switching events show in the transfer curve.
