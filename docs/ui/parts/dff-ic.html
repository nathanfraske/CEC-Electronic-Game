The D flip-flop, five layers down

A flip-flop is the smallest unit of memory: one bit that stays put until you tell it to change. This is the 74AUP1G79, a single positive-edge-triggered D-type flip-flop in a five-pin package. It does one thing. On each rising edge of the clock CP it samples the data input D and copies that value to the output Q, then holds Q steady until the next rising edge, no matter what D does in between. That hold is the memory. Internally it is a master-slave pair of latches clocked on opposite phases, so the value is captured at the instant the clock rises and only then released to Q, never letting input and output be connected at once. The first tier is the logic symbol with the real pinout and a live timing diagram. The second is the flow network, the two latches as chambers that hand a bit across on the clock edge. The third opens that mechanism as pressure-pilot valves. The fourth is the schematic, the transmission-gate master-slave built from inverters and clocked switches. The fifth is the silicon. The cross-coupled inverter pair at the heart of each latch is the storage cell, the same regenerative loop that holds a bit everywhere in a chip.
